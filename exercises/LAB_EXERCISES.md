# ELK Stack Lab Exercises

This document contains practical, hands-on scenarios to help you master the ELK Stack. Each exercise includes a problem statement, setup instructions, and a complete solution.

## Core Exercises
1. [Web Server Log Monitoring](./01-web-server-monitoring.md) - Monitor Nginx logs for 404 errors.
2. [Failed Login Detection](./02-failed-login-detection.md) - Identify SSH brute-force attacks.
3. [Application Error Tracking](./03-application-error-tracking.md) - Handle multi-line Java stack traces.
4. [Direct Data Ingestion](./04-direct-data-ingestion.md) - Push JSON directly to the Elasticsearch API.
5. [Centralized Logging with GELF](./05-centralized-logging-gelf.md) - Ingest structured logs via GELF.

## Advanced Topics
6. [Beats Data Collection](./06-beats-data-collection.md) - Use Filebeat for lightweight log shipping.
7. [Grok Advanced Parsing](./07-grok-advanced-parsing.md) - Parse custom log formats with complex Grok patterns.
